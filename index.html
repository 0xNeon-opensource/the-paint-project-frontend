The Paint Project

The Paint Project is a collection of the first colors as NFTs on any open public blockchain: created by the most important person in the metaverse: you.

The colors created here will reamain immutably on Ethereum, completely on-chain.

This is a community-centered project:

     
  • Since all data is stored on-chain, anyone can build anything on top of The Paint Project. Check out the Discord #Creators channel if you have an idea!
    •  
  • There will be a fair minting process, everyone gets access at the same time. Announcement will be made in the Twitter and Discord.
    •  
  • Mint price is 0. Users just pay gas.
    •  
  • There is no roadmap. This was a deliberate decision to let the community drive where it wants to take this project. The Paint Project creators' job is to organize the community, not decide its direction.

Infinite thanks to the contributors of JS Paint, the project that was forked to create this web page.

The code for this frontend and contract will be made open source after all 1024 colors are minted.
